Given materials of different value/weight ratios, find the most valuable combination of materials which fit in a knapsack of fixed capacity W. Greedy algorithm with max value/weight metric finds optimal solution for continuous knapsack problem. Optimal solution for 0/1 knapsack problem - formal problem statement: Given vectors of weight wi, and values vi, where 0 <= i <= N-1 items; knapsack holds W. Find 0/1 vector ti.
